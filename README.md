# 📦 Olist E-commerce Sales & Delivery Performance Analysis

## 📌 Project Goal
The goal of this project is to analyze Olist’s e-commerce data to understand sales performance, logistics efficiency, customer satisfaction, and revenue-driving categories, enabling data-driven business decisions across operations and customer experience.

---

## 📌 Business Problem
The business wants to answer:

- Which product categories generate the highest revenue?
- Which cities contribute the most orders and sales?
- How do delivery delays impact customer ratings?
- What payment behaviors are common across customers?
- Where can logistics performance be improved?

---

## 📂 Dataset
The dataset contains multiple relational tables including:

- Orders
- Customers
- Products
- Payments
- Reviews
- Sellers
- Order Items
- Geolocation

This project used a multi-table e-commerce relational dataset.

---

## 🛠 Tools Used
- SQL (MySQL) → Data cleaning, joins, KPI dataset creation
- Power BI → Dashboard development & DAX measures
- Excel / CSV → Raw data source
- GitHub → Documentation & project versioning

---

## 🔄 Workflow
1. Imported raw CSV files into MySQL
2. Performed data cleaning and date transformation
3. Joined multiple transactional tables
4. Created SQL analytical fact and summary tables
5. Engineered KPIs:
   - Revenue
   - Delivery Days
   - Late Delivery %
   - Average Review Score
6. Imported SQL-prepared dataset into Power BI
7. Built interactive dashboard for executive insights

---

## 📊 KPI Metrics
- Total Orders
- Total Revenue
- Average Delivery Days
- Late Delivery %
- Average Review Score
- Category Revenue
- City-wise Revenue
- Delivery Delay Impact

---

## 📈 Key Insights
- Delivery delays show a negative relationship with review scores
- A few categories drive the majority of revenue
- Revenue is concentrated in major cities
- Late delivery rate is controlled but still impacts ratings
- Average delivery timeline indicates logistics optimization opportunity

---

## 💡 Business Recommendations
- Improve SLA monitoring for delayed deliveries
- Focus inventory on high-performing categories
- Strengthen logistics in top revenue cities
- Use review score trends as customer experience KPI
- Optimize shipping routes for faster fulfillment
